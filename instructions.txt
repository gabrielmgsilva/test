INSTRUCTIONS:

The application is using h2 database located in a fisical file in the application.

To run the application use the following command in the project root folder:

java -jar target/demo-0.0.1-SNAPSHOT.war

ENDPOINTS:

(GET) /weather
(QUERY PARAMS) latitude | longitude

FULL REQUEST PATH EXAMPLE

http://localhost:8080/weather?latitude=-23.45&longitude=-46.52


RESPONSE:

{
	"weather": {
		"id": 38,
		"temperature": 26.6,
		"windspeed": 15.3,
		"winddirection": 297.0,
		"time": "2023-02-05T17:00:00",
		"utctime": "2023-02-05T17:00:00",
		"location": {
			"id": 38,
			"longitude": -46.52,
			"latitude": -23.45,
			"currentWeather": true,
			"weathers": null
		}
	}
}


The file is generated by default in the project root folder, by the name starting with "weather-table"


To access the database use the following URL:

http://localhost:8080/h2-console

user: sa

and it has no password. So leave it blank.
